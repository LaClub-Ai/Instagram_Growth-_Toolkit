A professional tool for safe and sustainable Instagram follower growth using methods that are fully compliant with Instagram rules.

## ✨ Features

- Intelligent analysis of hashtags and highly engaged posts
- Automatic engagement with safe limits
- Intelligent content scheduling
- Advanced management dashboard
- Reporting and performance analysis
- Alert system to avoid limits

## 🛠 Setup

### Prerequisites

- Python 3.9+
- Docker (for deployment)
- Access to Instagram API
